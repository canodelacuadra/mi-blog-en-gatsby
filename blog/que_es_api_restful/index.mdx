Una API REST (RESTful API) es un tipo de interfaz de programación de aplicaciones (API) que se basa en los principios de la arquitectura REST (Representational State Transfer o Transferencia de Estado Representacional). 
REST es un conjunto de reglas o pautas de diseño que definen cómo las aplicaciones web intercambian información a través de HTTP (Protocolo de Transferencia de Hipertexto).

```javascript
// Example using a public API (replace with your desired API endpoint)
const apiUrl = 'https://api.example.com/users';

// Fetch data using GET request
fetch(apiUrl)
  .then(response => {
    // Check for successful response
    if (!response.ok) {
      throw new Error(`Error fetching data: ${response.status}`);
    }
    return response.json(); // Parse JSON response
  })
  .then(data => {
    // Process the fetched data (users in this case)
    console.log('Fetched user data:', data);
  })
  .catch(error => {
    console.error('Error:', error);
  });
  ```
