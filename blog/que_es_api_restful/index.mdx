Una API REST (RESTful API) es un tipo de interfaz de programación de aplicaciones (API) que se basa en los principios de la arquitectura REST (Representational State Transfer o Transferencia de Estado Representacional). 
REST es un conjunto de reglas o pautas de diseño que definen cómo las aplicaciones web intercambian información a través de HTTP (Protocolo de Transferencia de Hipertexto).
