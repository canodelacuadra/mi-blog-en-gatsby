GraphQL es un lenguaje de consulta de datos y un entorno de ejecución para APIs. 
A diferencia de REST que define cómo transferir información,
 GraphQL se centra en cómo especificar de forma exacta los datos que necesitas.

 ```graphql
query ($id: String) {
  mdx(id: {eq: $id}) {
    frontmatter {
      title
      date(formatString: "MMMM D, YYYY")
      hero_image_alt
      hero_image_credit_link
      hero_image_credit_text
      hero_image {
        childImageSharp {
          gatsbyImageData
        }
      }
    }
  }
}
 
 ```
